- Select vraća samo naziv kina.

- Any znači “postoji li barem jedan element koji zadovoljava uvjet”.

Što radi:
- Filtrira ulaznice koje imaju status Active.
- Vraća samo TicketNumber.

Logika:
- Where radi filtriranje.
- Select radi projekciju (uzima samo jedno polje iz objekta).

3. screeningsAfter19  
Kod: Main.cs

- Iz kina ide u dvorane, pa iz dvorana u projekcije.
- Zadržava samo projekcije koje počinju od 19:00 nadalje.
- Formatira rezultat u tekst “Film (datum vrijeme)”.

Logika:
- SelectMany “spljošti” hijerarhiju:
- prvo List<Cinema> -> sve dvorane,
- zatim sve dvorane -> sve projekcije.
- Nakon toga Where i Select rade kao i inače.

4. topLoyaltyCustomer  
Kod: Main.cs  
Što radi:
- Sortira kupce silazno po LoyaltyPoints.
- Uzimа prvog iz sortiranog niza (onaj s najviše bodova).
- Ako je kolekcija prazna, FirstOrDefault vraća null.

Logika:
- Zato kasnije imaš null provjeru prije ispisa.

Korisna napomena:
- Prva tri upita se stvarno izvršavaju tek u foreach petljama kod ispisa (deferred execution).
- topLoyaltyCustomer se izvrši odmah jer FirstOrDefault traži konkretan element (terminal operation).
